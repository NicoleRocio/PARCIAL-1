Routing & Consume API - Laboratorio 03

Front-end Routing

En esta sección, aprenderemos sobre el enrutamiento en el lado del cliente (Front-end Routing) utilizando React-Router. El enrutamiento nos permite navegar entre diferentes páginas sin recargar la página completa, proporcionando una experiencia de usuario más fluida.

Promesas y Async/Await

Exploraremos cómo trabajar con Promesas y Async/Await en JavaScript. Estas características nos permiten manejar operaciones asincrónicas de manera más sencilla y legible, evitando el anidamiento excesivo de callbacks.

Obteniendo datos de una API

Aprenderemos cómo realizar peticiones a una API desde el Front-end utilizando herramientas como Axios o fetch. Podremos obtener datos de un servidor y mostrarlos en nuestra interfaz de usuario.

Laboratorio 03: Caso práctico consumo de API – Parte 1

En este laboratorio, trabajaremos en un caso práctico de consumo de API. Comenzaremos creando el proyecto, configurando las dependencias necesarias y creando las rutas para nuestra aplicación Front-end.

Componentes Formulario

Crearemos componentes de formulario reutilizables para capturar datos del usuario. Aprenderemos a manejar eventos, validar datos y enviar formularios a través de API.
