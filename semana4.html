Manejo del DOM con Javascript

Métodos y Propiedades

El DOM (Document Object Model) es una representación en memoria de la estructura HTML de una página web. Aprenderemos cómo utilizar métodos y propiedades de Javascript para acceder, manipular y modificar los elementos del DOM de una manera dinámica y eficiente.

Elementos / Estilos con Javascript

Con Javascript, podemos crear, modificar y eliminar elementos del DOM de forma dinámica. También exploraremos cómo cambiar los estilos y atributos de los elementos utilizando Javascript, lo que nos permite crear interacciones y efectos visuales interesantes en nuestras aplicaciones web.

Animaciones

Las animaciones en una página web pueden agregar vida y atractivo visual. Aprenderemos cómo crear animaciones utilizando Javascript y CSS para crear transiciones suaves, efectos de desplazamiento y otros tipos de animaciones que mejoren la experiencia del usuario.

Trabajo 03: Desarrollo de un juego con Javascript

En este trabajo, pondrás en práctica tus habilidades de programación con Javascript. Desarrollarás un juego interactivo utilizando el DOM, eventos y animaciones. Aprenderás a gestionar la lógica del juego, manejar eventos del teclado o del ratón, y crear una experiencia de juego divertida y desafiante.
