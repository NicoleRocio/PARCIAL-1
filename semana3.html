Programación Front-End Javascript

Lenguaje Javascript

Javascript es un lenguaje de programación ampliamente utilizado en el desarrollo web. Aprenderemos sobre los conceptos básicos de Javascript, incluyendo números, strings, objetos y métodos. También exploraremos el uso de booleanos, funciones y funciones flecha.

Gráficos

Con Javascript, podemos crear gráficos y visualizaciones dinámicas en el navegador. Exploraremos bibliotecas populares como D3.js y Chart.js para crear gráficos interactivos y atractivos que mejoren la experiencia del usuario en el front-end.

Trabajo 02: Desarrollo de SVG Logo FIS

En este trabajo, pondrás en práctica tus habilidades de programación front-end. Desarrollarás un logo interactivo para FIS utilizando SVG (Scalable Vector Graphics). Aprenderás a manipular elementos SVG, animarlos y agregar interactividad a través de eventos de Javascript.
