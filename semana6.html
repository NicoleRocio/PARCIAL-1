Eventos, Condicionales y Bucles con Framework JS

Eventos

En el desarrollo con Frameworks JS, trabajamos con eventos para capturar y manejar interacciones del usuario, como clics, cambios en formularios, movimientos del mouse, entre otros. Aprenderemos cómo utilizar eventos en nuestro código y cómo vincular funciones para responder a ellos de manera adecuada.

Renderizado condicional / Renderizado iterativo

En el desarrollo de aplicaciones con Frameworks JS, es común realizar renderizado condicional y renderizado iterativo. Aprenderemos cómo mostrar u ocultar elementos en función de condiciones lógicas, así como generar contenido repetitivo utilizando bucles o iteradores proporcionados por el Framework.

Formularios

Los formularios son una parte fundamental de muchas aplicaciones web. Aprenderemos cómo trabajar con formularios en el contexto de un Framework JS, cómo capturar datos ingresados por el usuario y cómo manejar validaciones y envíos de formularios de manera eficiente.

Eventos y Renderizado Condicional e iterativo

En este laboratorio, pondremos en práctica los conceptos aprendidos. Aprenderemos a trabajar con eventos en un proyecto de Framework JS, así como a realizar renderizado condicional e iterativo para construir una interfaz dinámica y reactiva.

Utilización de Estilos en Framework JS

Los Frameworks JS ofrecen diversas formas de aplicar estilos a nuestros componentes. Exploraremos opciones como estilos inline, hojas de estilos externas, módulos de estilos y componentes de estilos proporcionados por el Framework o una biblioteca externa.

Styles components / Framework CSS

En el desarrollo con Frameworks JS, también podemos utilizar librerías como styled-components o utilizar Frameworks CSS como Bootstrap o Material UI para facilitar la definición y aplicación de estilos en nuestros componentes.

Laboratorio 02B: Uso de estilos

En este laboratorio, pondremos en práctica los conceptos de estilos aprendidos. Aprenderemos a aplicar estilos en un proyecto de Framework JS utilizando diferentes enfoques, como estilos inline, hojas de estilos externas y componentes de estilos proporcionados por una biblioteca o Framework.
