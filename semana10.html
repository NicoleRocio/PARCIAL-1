Uso de Hooks - Laboratorio 04

UseState

Utilizaremos el Hook UseState para manejar el estado de nuestros componentes funcionales. Esto nos permitirá almacenar y actualizar datos a lo largo del ciclo de vida del componente.

UseEffect

Con el Hook UseEffect, podremos realizar efectos secundarios en nuestros componentes, como llamadas a API, suscripciones a eventos y manipulación del DOM. También aprenderemos a limpiar estos efectos cuando el componente se desmonte.

UseContext

El Hook UseContext nos permitirá compartir datos entre componentes sin necesidad de pasar props manualmente. Crearemos y consumiremos un contexto para nuestra aplicación.

UseRef

El Hook UseRef nos permitirá hacer referencia a elementos del DOM o a valores que persisten durante toda la vida del componente, sin causar una nueva renderización.

UseReducer

Con el Hook UseReducer, podremos manejar estados complejos y lógica de actualización en nuestros componentes, de manera similar a Redux, pero a nivel de componente.

UseCallback

Utilizaremos el Hook UseCallback para optimizar el rendimiento de nuestros componentes, memorizando funciones para que no se creen en cada renderización.

UseMemo

Con el Hook UseMemo, podremos optimizar el rendimiento de cálculos costosos, memorizando el resultado para que solo se recalcule cuando sus dependencias cambien.

Hooks Personalizados

Aprenderemos cómo crear nuestros propios hooks personalizados para reutilizar lógica compleja en diferentes componentes.
