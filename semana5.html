Desarrollo FrontEnd con Framework JS

Estructura y operación de Framework – Client Side Rendering

Los Frameworks de JavaScript, como React o Vue, permiten el desarrollo de aplicaciones web modernas utilizando el enfoque de renderizado en el lado del cliente. Exploraremos la estructura y el funcionamiento de estos Frameworks, y cómo se realiza el renderizado de componentes en el navegador.

Gestión de dependencias

En el desarrollo con Frameworks JS, es común utilizar herramientas de gestión de dependencias como npm o Yarn. Aprenderemos cómo gestionar las dependencias de nuestro proyecto, instalar paquetes externos y mantener un control adecuado de las versiones de las librerías utilizadas.

Herramientas de creación de proyectos

Existen diversas herramientas que facilitan la creación y configuración inicial de proyectos con Frameworks JS. Exploraremos herramientas populares como Create React App o Vue CLI, que nos permiten generar rápidamente una estructura de proyecto con todas las configuraciones necesarias para comenzar a desarrollar.

Laboratorio 01A: Instalación del proyecto y configuración de dependencias

En este laboratorio, pondremos en práctica los conceptos aprendidos. Aprenderemos a instalar un proyecto con el Framework JS seleccionado, configurar las dependencias necesarias y realizar una configuración básica para comenzar a desarrollar nuestra aplicación web.

Desarrollo basado en componentes – JSX / PROPS / CHILDREN

En el desarrollo con Frameworks JS, utilizamos un enfoque basado en componentes. Aprenderemos cómo utilizar JSX para definir componentes de manera declarativa, cómo pasar propiedades (props) a los componentes y cómo trabajar con el concepto de children para anidar componentes dentro de otros.
