Sílabo

En esta exposición, presentaremos el sílabo del curso de Fundamentos de las Tecnologías Web. Aprenderemos los conceptos básicos de HTML, CSS y JavaScript, y exploraremos cómo crear aplicaciones para Internet tanto en el desarrollo front-end como en el desarrollo back-end.

Fundamentos de las Tecnologías Web

En esta sección, exploraremos los fundamentos de las tecnologías web. Aprenderemos a estructurar páginas web con HTML, dar estilo a través de CSS y agregar interactividad utilizando JavaScript. ¡Prepárate para sumergirte en el mundo del desarrollo web!

Aplicaciones para Internet

En esta parte del curso, nos enfocaremos en el desarrollo de aplicaciones para Internet. Aprenderemos a crear interfaces de usuario atractivas utilizando las mejores prácticas de desarrollo front-end. También exploraremos cómo trabajar con bases de datos y desarrollar la lógica del lado del servidor en el desarrollo back-end.

Mockups / UX Design

El diseño de experiencia de usuario (UX) es fundamental para crear aplicaciones web exitosas. En esta sección, aprenderemos sobre la creación de mockups y cómo diseñar interfaces de usuario intuitivas y agradables. ¡Descubriremos cómo hacer que los usuarios amen nuestras aplicaciones desde el primer vistazo!
